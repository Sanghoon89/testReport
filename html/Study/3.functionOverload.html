함수 오버로딩과 타입 검사

자바스크립트 함수 오버로딩에 관한 두 가지 예

    함수 오버로딩에 필요한 두가지 정보 : 전달인자 개수와 전달인자 타입
  1. sendMessage(msg, obj) : 메시지를 보내는 간단한 함수
    • sendMessage("Hello, World!");
    • sendMessage("How are you?", window);

      • 두 호출 모두 제대로 작동

  2. makeArray() : 전달인자를 여러 개 받아 배열을 만들어 반환하는 함수
  • displayError(msg) : 에러 메시지나 기본 메시지 출력하기
    • displayError();
    • displayError('This message is OK.');

변수의 타입을 검사하는 두가지 방법

  1. typeof를 사용한 예
  2. constructor 프로퍼티를 사용한 예

함수에 전달되는 모든 전달인자를 엄격하게 관리하는 함수

  1. 변수 타입 목록을 기준으로 전달인자 목록을 엄격히 검사
  2. 사용자 목록을 출력하는 간단한 함수 (위 함수 활용)
